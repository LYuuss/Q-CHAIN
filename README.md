# Q-Chain

Q-Chain is an experimental Proof-of-Work blockchain designed as a foundation for post-quantum cryptography and future zero-knowledge privacy layers.

## Current features

- Proof of Work
- SHA3-256 hashing
- Wallets
- Signed transactions
- Signature verification
- Balances
- Nonces
- Mining reward
- Cached state
- Mempool
- Mempool double-spend protection  

## Missing features 

- Quantum proofness
- Handling forks
- P2P communication
